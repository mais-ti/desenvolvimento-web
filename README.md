## Aulas

| Aula | Conteúdo                                                                                                                                   |
| :--: | :----------------------------------------------------------------------------------------------------------------------------------------- |
|   1  | Arquitetura cliente-servidor, introdução ao desenvolvimento web, HTTP, estrutura básica do HTML5, textos, títulos, links e navegação.      |
|   2  | HTML avançado: listas, imagens, tabelas, formulários, validação, elementos semânticos e editores WYSIWYG.                                  |
|   3  | Fundamentos de CSS: seletores, cascata, especificidade, unidades de medida, cores, tipografia, box model e posicionamento.                 |
|   4  | Layouts responsivos com Flexbox e Grid, pseudo-classes, pseudo-elementos, animações, transições, media queries e variáveis CSS.            |
|   5  | JavaScript Frontend: manipulação do DOM, eventos, formulários, classes, estilos, criação dinâmica de elementos e eventos de teclado/mouse. |
|   6  | JavaScript avançado: animações com JS, consumo de APIs REST com Fetch, JSON, tratamento de erros, LocalStorage e SessionStorage.           |
|   7  | Bibliotecas externas: Bootstrap, Tailwind CSS e jQuery.                                                                                    |
|   8  | APIs externas: Axios, Web APIs, Fetch API, Geolocation API, armazenamento local e integração com APIs públicas.                            |
|   9  | Node.js: ambiente backend, módulos, npm, servidor HTTP, Express.js, APIs REST e persistência de dados.                                     |
|  10  | Express.js na prática: rotas, parâmetros, query strings, middlewares, JSON, integração com MySQL e testes via Postman/Insomnia.            |
|  11  | Projeto integrador: especificação e implementação do backend de um sistema de pedidos utilizando Node.js, Express e MySQL.                 |
|  12  | Projeto integrador: desenvolvimento do frontend, integração entre equipes utilizando GitHub e conclusão do sistema completo.               |

## Competências Desenvolvidas

* Desenvolvimento de páginas web estáticas com HTML e CSS.
* Criação de interfaces responsivas.
* Programação frontend com JavaScript.
* Consumo de APIs REST.
* Desenvolvimento backend com Node.js e Express.
* Implementação de CRUD com banco de dados MySQL.
* Estruturação de aplicações web seguindo arquitetura cliente-servidor.
* Integração de projetos utilizando Git e GitHub.
* Construção de aplicações web completas (frontend + backend).

## Projeto da Disciplina

Ao longo das aulas será desenvolvido um sistema web de pedidos para restaurante, utilizado como projeto integrador para aplicação dos conceitos de frontend, backend, APIs e banco de dados.
